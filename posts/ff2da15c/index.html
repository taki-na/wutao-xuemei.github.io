准备工作

找到设备ID

  1. 打开warp软件（1.1.1.1）
  2. 点击右上角菜单键
  3. 高级>诊断
  4. 在“客户端配置”下>复制 ID

下载软件

下载链接

使用方法

  1. 解压压缩包
  2. 打开"dist"文件夹
  3. 运行"wp-plus.exe"
  4. 输入你的设备ID
